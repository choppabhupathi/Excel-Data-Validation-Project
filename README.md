# Excel Data Validation Project 📊

This project demonstrates how to use **Data Validation** in Microsoft Excel to maintain data integrity and prevent manual entry errors. I have applied 5 essential validation rules to a sample Employee Database.

## 🚀 Features & Validations

I have implemented the following validation techniques:

1. **Drop-down List (List Validation):** - Created a selection menu for 'Employee Names' to ensure no spelling mistakes.
2. **Whole Number Validation:** - Restricted the 'Salary' column to accept only values greater than 10,000.
3. **Text Length Validation:** - Set a rule for 'Employee Code' to accept exactly 6 characters (e.g., DS-101).
4. **Date Validation:** - Restricted 'Joining Date' to accept only dates from January 1, 2020, onwards.
5. **Decimal Validation:** - Added a 'Performance Rating' column that only accepts decimal values between 1.0 and 10.0.
